# Strava Segments Viewer

https://colscoding.github.io/segments/

A web application for browsing and filtering Strava running segments.

## Project Structure

```
public/          # Source files for the web app
docs/            # Production build (GitHub Pages)
```

## Scripts

- `npm run serve` - Start development server with Parcel
- `npm run start:server` - Start the API server
- `npm start` - Run both dev server and API server
- `npm run build` - Build for production to docs/
- `npm run serve:docs` - Preview production build
